About Us

Welcome to our Weather App! This application provides real-time weather updates for cities around the world. You can easily search any city and get accurate information on temperature, humidity, wind speed, and weather conditions.

Features:

  • Search cities worldwide
  • View temperature, humidity, wind, and weather condition
  • Dynamic weather images/icons based on real-time data
  • User-friendly interface for easy navigation

Developed by: Yash Kumar

Built with: HTML, CSS, JavaScript, and OpenWeatherMap API
